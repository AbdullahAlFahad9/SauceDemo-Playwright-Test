# SauceDemo Playwright Test

This repository contains a Playwright automated test for the SauceDemo website.  
The test performs the following steps:

1. Opens the SauceDemo website.
2. Logs in with valid credentials.
3. Adds the first product to the cart.
4. Verifies the product name in the cart.
5. Logs out.
